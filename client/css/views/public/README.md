# Public CSS

Styles for public-facing views.

## Files

| File         | Description                                                 |
| ------------ | ----------------------------------------------------------- |
| `public.css` | Public page layouts (homepage, maps)                        |
| `auth.css`   | Authentication page styles (login, signup, forgot-password) |

## Usage

```html
<!-- Public pages -->
<link rel="stylesheet" href="../../../css/views/public/public.css" />

<!-- Auth pages -->
<link rel="stylesheet" href="../../../css/views/public/auth.css" />
```
