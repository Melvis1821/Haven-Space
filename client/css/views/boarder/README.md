# Boarder CSS

Styles for boarder dashboard views.

## Files

| File                       | Description                   |
| -------------------------- | ----------------------------- |
| `boarder.css`              | Boarder dashboard home styles |
| `boarder-rooms.css`        | Room browsing styles          |
| `boarder-applications.css` | Rental application styles     |
| `boarder-payments.css`     | Payment management styles     |
| `boarder-maintenance.css`  | Maintenance request styles    |

## Usage

```html
<!-- Dashboard home -->
<link rel="stylesheet" href="../../../css/views/boarder/boarder.css" />

<!-- Rooms -->
<link rel="stylesheet" href="../../../css/views/boarder/boarder-rooms.css" />

<!-- Applications -->
<link rel="stylesheet" href="../../../css/views/boarder/boarder-applications.css" />

<!-- Payments -->
<link rel="stylesheet" href="../../../css/views/boarder/boarder-payments.css" />

<!-- Maintenance -->
<link rel="stylesheet" href="../../../css/views/boarder/boarder-maintenance.css" />
```
